vim-machine
=========

This role installs and configures Vim.

Requirements
------------

This role depends on package manager. For Linux it is [APT](https://wiki.debian.org/Apt) and for OS X it's [Homebrew](http://brew.sh).

Example Playbook
----------------

    - hosts: stage
      roles:
        - vim-machine
